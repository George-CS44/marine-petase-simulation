# Overview - Computer simulated project of modified cold adapted PETase enzyme kinetics to reduce ocean plastic pollution

Idea behind the project-
Plastic Pollution in the ocean is a major problem with no long term sollutions at the moment. This simulation presents a curretnly theoretical solution by adding a modified PETase enzyme into the ocean that is able to break down the polyethylene terephthalate in the conditions that conventional PETase is hindered by.
